# Installation Guide

## npm (Primary)

```bash
npm install -g @talocode/gatelane
```

## npx

```bash
npx @talocode/gatelane demo
```

## Linux

```bash
npm install -g @talocode/gatelane
gatelane demo
```

## macOS

```bash
npm install -g @talocode/gatelane
gatelane demo
```

## Windows

```bash
npm install -g @talocode/gatelane
gatelane demo
```

## Android (Termux)

```bash
pkg install nodejs
npm install -g @talocode/gatelane
gatelane demo
```

## iOS (iSH)

```bash
apk add nodejs npm
npm install -g @talocode/gatelane
gatelane demo
```

Note: iOS support is via iSH shell (Node-compatible environment). No native iOS binary is provided in v0.1.0. For remote SDK usage, point GateLaneClient at a server running on your infrastructure.

## Requirements

- Node.js >= 18.0.0
- npm (included with Node.js)
